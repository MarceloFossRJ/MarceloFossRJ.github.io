# Markdown Cheatsheet
- - - -

# Heading
```
  # H1
  ## H2
  ### H3
  #### H4
  ##### H5
  ###### H6
```

## Emphasized/Italic text
```
    _Emphasized text_ or *Emphasized text*
```
_Emphasized text_

## Strikethrough text
```
    ~~Strikethrough text~~
```
~~Strikethrough text~~

## Strong/Bold text
```
    __Strong text__  
    **Strong text**
```
__Strong text__

## Strong emphasized text
```
    ___Strong emphasized text___   
    ***Strong emphasized text***
```
___Strong emphasized text___

## Links

```
    [Named Link](http://www.marcelofossrj.com/)  
    http://www.marcelofossrj.com/  
    <http://www.marcelofossrj.com/>
```
[Named Link](http://www.marcelofossrj.com/)   
http://www.marcelofossrj.com/  
<http://www.marcelofossrj.com/>  

To open a link in a new window
```
[Named Link](http://www.marcelofossrj.com/){:target="_blank"}
```

## Tables
```
First Header  | Second Header

```

## Bullet list

```
 * Bullet list
    * Nested bullet
        * Sub-nested bullet etc
 * Bullet list item 2
```
* Bullet list
    * Nested bullet
        * Sub-nested bullet etc
* Bullet list item 2


## Numbered list
```
 1. A numbered list
      1. A nested numbered list
      2. Which is numbered
 2. Which is numbered
```

1. A numbered list
    1. A nested numbered list
    2. Which is numbered
2. Which is numbered


## Check box
```
 - [ ] An uncompleted task
 - [x] A completed task
```
- [ ] An uncompleted task
- [x] A completed task
